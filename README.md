A fun, educational mushroom identification game focused on Michigan foraging!

## Play Now

Just open `index.html` in any browser - no build step required!

Or host it anywhere (GitHub Pages, Netlify, etc.)

## Features

- **16 Michigan mushrooms** with detailed descriptions
- **Educational facts** about each species
- **Safety-focused** - emphasizes the "when in doubt, throw it out" rule
- **Difficulty levels** - Easy (obvious ones) vs Hard (tricky lookalikes)
- **Score tracking** with streak bonuses
- **Mobile-friendly** responsive design
- **Beautiful UI** with dark forest theme
